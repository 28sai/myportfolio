My Projects

Credit Card Approval Prediction System

This project is about predicting whether a credit card application will be approved using data analysis and machine learning. By looking at details like income, credit history, and job status, the model helps banks make better decisions. It shows how data can make the approval process faster and more reliable.

Tourism

This project is about promoting Telangana's tourism by creating a user-friendly website. The site showcases the state's attractions, culture, and heritage, making it easier for tourists to plan their visits. Sculpture Prediction

This project is about predicting and identifying two specific sculptures, the pillar from the Thousand Pillar Temple and the Nandi from the Ramappa Temple. Using image recognition techniques, the model accurately distinguishes these sculptures, showcasing how technology can be used to preserve and study cultural heritage.
